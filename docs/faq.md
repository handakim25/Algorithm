# ❓ FAQ — 자주 묻는 질문

알고리즘 학습 중 자주 나오는 질문들을 정리했습니다. 😊

---

## Q1. 30분 넘으면 무조건 힌트를 봐야 하나요?

**A**: 아니요! 30분은 절대적인 기준이 아니라 **체크포인트**입니다. 🎯

30분이 됐을 때 가장 중요한 건 **진전 여부를 스스로 점검**하는 것입니다.

- ✅ **진전이 있다면** → 더 고민해보세요! 접근법을 찾았거나 새로운 아이디어가 떠오르고 있다면 계속 진행하세요.
- 🚫 **완전히 막혔다면** → 힌트를 확인하는 것이 더 효율적입니다. 같은 생각만 반복하는 상태에서 시간을 더 쓰는 건 학습 효과가 낮아요.

> 💡 자세한 기준은 [시간 관리 전략 문서](./time-management.md#-진전-상황-판단-기준)를 참고하세요!

---

## Q2. 힌트를 보면 실력이 안 늘지 않나요?

**A**: 힌트를 **어떻게 활용하느냐**가 핵심입니다! 🔑

힌트를 보고 그냥 넘어가면 실력이 잘 안 늘지만, 제대로 활용하면 오히려 **실력 향상에 큰 도움**이 됩니다.

### ✅ 올바른 힌트 활용법

1. **힌트만 보고 창 닫기** — 접근법만 파악하고 바로 스스로 구현
2. **복붙 금지** — 솔루션 코드를 봤다면 직접 타이핑하며 이해
3. **주석 달기** — 각 줄이 왜 그렇게 동작하는지 스스로 설명
4. **다음날 재풀이** — 백지 상태에서 다시 풀어보는 것이 핵심!
5. **1주일 후 재풀이** — 장기 기억으로 전환

> 🧠 힌트를 본 후에도 "다음날 재풀이"를 실천하면 실력이 충분히 향상됩니다!

---

## Q3. 너무 오래 고민하는 게 습관인데 어떻게 하나요?

**A**: 타이머를 적극 활용하고 체크포인트 습관을 만들어보세요! ⏰

### 추천 방법

1. **타이머 설정**: 처음부터 목표 시간을 설정해두기
   - Level 3 기준: 30분 타이머 설정 → 알림 울리면 진전 여부 체크
2. **체크리스트 활용**: 막힌 상태인지 판단하는 기준을 체크리스트로 보기
3. **학습 로그 기록**: 각 문제에 소요된 시간을 기록하면 습관 파악에 도움

### 오래 고민하는 것의 득과 실

| | 득 | 실 |
|--|---|---|
| **오래 고민** | 사고력 향상, 성취감 | 시간 비효율, 번아웃 위험 |
| **적절한 힌트** | 새로운 패턴 학습, 효율적 시간 사용 | 스스로 발견하는 경험 감소 |

> 💡 **균형이 중요합니다!** 막힌 상태에서 3시간을 쓰는 것보다 1시간 고민 후 힌트를 보고 다음날 재풀이하는 것이 훨씬 효율적입니다.

## Q4. 실전 코딩테스트에서는 어떻게 시간 관리를 하나요?

**A**: 실전에서는 **학습 모드와 전혀 다른 전략**이 필요합니다! 🏆

### 실전 코딩테스트 시간 전략

| 단계 | 전략 |
|------|------|
| **문제 확인** (5분) | 모든 문제를 먼저 훑어보고 쉬운 것부터 파악 |
| **쉬운 문제** 먼저 | 확실히 풀 수 있는 문제부터 빠르게 처리 |
| **중간 문제** | 30분 내로 접근법이 보이지 않으면 다음 문제로 |
| **어려운 문제** | 시간이 남으면 도전, 부분 점수라도 획득 |
| **막힌 문제** | 메모만 해두고 다른 문제 먼저 완료 |

### 실전 대비 연습법

- **모의 테스트**: 시간 제한 엄수 (힌트 없이!)
- **시험 1~2달 전**: 30분/문제 기준으로 엄격하게 연습
- **테스트 후**: 못 푼 문제 반드시 복습

> ⚠️ **주의**: 실전에서는 힌트가 없습니다! 평소에 다양한 유형을 충분히 학습해두는 것이 중요합니다.

---

## Q5. 하루에 몇 문제를 풀어야 하나요?

**A**: 양보다 **질**이 중요합니다! 🌟

| 레벨 | 권장 문제 수 | 중요 포인트 |
|------|------------|------------|
| 입문~초급 (Level 1-2) | 1~2문제/일 | 개념 이해 + 복습 |
| **중급 (Level 3)** | **2~3문제/일** | **깊이 이해 + 재풀이** |
| 고급 (Level 4-5) | 3~5문제/일 | 다양한 유형 + 최적화 |

> 💪 하루에 5문제를 빠르게 푸는 것보다, **2~3문제를 제대로 이해하고 다음날 재풀이**하는 것이 실력 향상에 훨씬 효과적입니다!
