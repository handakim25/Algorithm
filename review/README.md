## 📚 복습 전략

### 에빙하우스 망각 곡선 활용
학습 직후에는 기억이 잘 유지되지만, 시간이 지남에 따라 급격히 망각됩니다.

```
학습 직후 (100%)
   ↓ 1일 후 (58%)
   ↓ 7일 후 (33%)
   ↓ 30일 후 (21%)
   ↓ ...
```

### 최적 복습 주기
| 복습 회차 | 시점 | 방법 |
|---------|------|------|
| 1차 복습 | 문제 풀이 당일 | 접근 방법, 핵심 아이디어 정리 |
| 2차 복습 | 1일 후 | 핵심 아이디어 확인 |
| 3차 복습 | 7일 후 | 문제 다시 풀어보기 |
| 4차 복습 | 30일 후 | 유사 문제로 응용 확인 |

### 복습 우선순위
1. 🔴 **반드시 복습**: 못 풀었던 문제, 힌트/솔루션 참고한 문제
2. 🟡 **권장 복습**: 시간이 오래 걸린 문제 (목표 시간 2배 초과)
3. 🟢 **선택 복습**: 새로운 패턴으로 풀 수 있는 문제

## 📅 복습 스케줄

### 주간 복습 루틴
```
월요일: 지난 주 틀린 문제 오답 분석
화요일: 7일 전 문제 재풀이
수요일: 새로운 유형 학습 + 비슷한 유형 복습
목요일: 이번 주 어려웠던 문제 다시 풀기
금요일: 주간 복습 정리 + 다음 주 복습 계획
주말:   자유 복습 + 대회 참여
```

### 분야별 복습 주기 (중급자 기준)
| 분야 | 복습 주기 | 이유 |
|------|---------|------|
| 동적 프로그래밍 | 매주 | 가장 중요한 분야 |
| 그래프 심화 | 격주 | 다양한 유형 |
| 이진 탐색 | 격주 | 파라메트릭 서치 |
| 백트래킹 | 월 1회 | 패턴 반복 |
| 수학/기하 | 월 1회 | 특수 분야 |
