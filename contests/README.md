## 📊 플랫폼별 티어/레벨 시스템

### 백준 (BOJ) - Solved.ac 기준
| 티어 | 레벨 | 기준 |
|------|------|------|
| 🥉 Bronze (1-5) | 입문 | 기초 구현, 완전 탐색 |
| 🥈 Silver (1-5) | 초급 | 기본 알고리즘 (DFS/BFS, 정렬, DP 기초) |
| 🥇 Gold (1-5) | 중급 | 그래프, DP 심화, 이진탐색 |
| 🏆 Platinum (1-5) | 고급 | 고급 자료구조, 수학, 최적화 |
| 💎 Diamond (1-5) | 최상위 | 매우 어려운 문제 |
| 🔴 Ruby (1-5) | 전문가 | 최고난도 |

**현재 목표: 골드 3-5**

### 프로그래머스
| 레벨 | 기준 |
|------|------|
| Level 0 | 기초 |
| Level 1 | 입문 (완전 탐색, 기초 구현) |
| Level 2 | 초급 (DP, 스택/큐, 힙) |
| Level 3 | 중급 (그래프, DP 심화, 고급 탐색) |
| Level 4 | 고급 (고급 알고리즘) |
| Level 5 | 최상급 |

**현재: Level 3 일부 가능 → 목표: Level 3 완전 정복**

## 🎯 월간/분기별 목표

### 월간 목표 템플릿
```markdown
## YYYY년 MM월 목표

### 대회 참여 목표
- [ ] AtCoder ABC X회 참여
- [ ] Codeforces Div.2 X회 참여
- [ ] LeetCode Weekly X회 참여

### 문제 풀이 목표
- [ ] 프로그래머스: X문제
- [ ] 백준: X문제 (골드 이상)
- [ ] LeetCode: X문제 (Medium X, Hard X)

### 알고리즘 학습 목표
- [ ] [분야명] 개념 학습 완료
- [ ] [분야명] 추천 문제 완료

### 이번 달 회고
- 잘한 점:
- 개선할 점:
- 다음 달 집중 분야:
```
